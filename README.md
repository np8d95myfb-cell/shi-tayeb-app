# Shi Tayeb Full App

تطبيق محلي كامل للمستخدم، وليس مجرد صفحة عرض. يحتوي على:

- إنشاء فيديو من صورة + وصف + ستايل
- مكتبة جاهزة: وصفات ستي / مطبخ فاخر / وصفات صحية
- فيديوهات إعلانية جاهزة داخل التطبيق
- تسجيل دخول وإنشاء حساب
- باقات ودفع
- صفحة "من نحن"
- سجل فيديوهات

## التشغيل

### 1) تشغيل الخادم الخلفي
```bash
cd backend
npm install
npm start
```

### 2) فتح الواجهة
افتحي الملف:
`frontend/index.html`

## الدفع عبر PayPal
**مهم:** الإيميل وحده لا يكفي لإنشاء Checkout تلقائي كامل من PayPal داخل التطبيق.

لهذا السبب التطبيق الآن يعمل بطريقتين:
1. **إذا أضفتِ رابط دفع PayPal حقيقي** في `backend/.env` داخل `PAYPAL_PAYMENT_LINK`، سيفتحه التطبيق مباشرة للمستخدم.
2. **إذا تركتِه فارغاً**، سيستخدم التطبيق زر تواصل/طلب اشتراك عبر البريد إلى:
   `Loren.musa11@gmail.com`

### لتفعيل رابط دفع مباشر
انسخي `backend/.env.example` إلى ملف جديد اسمه `.env` ثم ضعي:
```env
PAYPAL_PAYMENT_LINK=https://www.paypal.com/...
PAYPAL_RECEIVER_EMAIL=Loren.musa11@gmail.com
```
